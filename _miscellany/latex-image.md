# LaTeX images 

Here is my snippet that I use when I want to place an image in a LaTeX document. 

```latex
\begin{center}
\begin{figure}[H]
\centering
\includegraphics[width=0.55\textwidth]{images/your-image.png}
\caption{Your text caption.}     
\end{figure}
\end{center}
```
